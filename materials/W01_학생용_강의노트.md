## 1. 변수(Variable) — 컴퓨터의 메모지

### 변수란 무엇인가

변수(Variable)란 **값을 저장하는 이름표**이다.

마치 서랍에 라벨을 붙이는 것과 같다.

```
서랍 라벨: "나이"      --> 서랍 안: 25
서랍 라벨: "이름"      --> 서랍 안: "홍길동"
서랍 라벨: "키"        --> 서랍 안: 175.3
```

파이썬에서는 이렇게 쓴다.

```python
나이 = 25
이름 = "홍길동"
키 = 175.3
```

등호(`=`)는 수학의 "같다"가 **아니다**. 파이썬에서 `=`는 **오른쪽 값을 왼쪽 이름표에 넣어라**는 명령이다.

---

### 변수 이름 짓기 규칙

변수 이름에는 규칙이 있다. 어기면 파이썬이 오류를 낸다.

| 규칙 | 예시 |
|:---|:---|
| 영문자, 숫자, 밑줄(`_`)만 사용 | `age`, `patient_id`, `score1` |
| 숫자로 시작 불가 | `1score` --> **오류!** |
| 공백 불가 | `my score` --> **오류!** |
| 파이썬 예약어 사용 불가 | `if`, `for`, `class` --> **오류!** |
| 대소문자 구별 | `Age`와 `age`는 **다른 변수** |

**좋은 변수 이름의 조건:** 읽었을 때 무슨 값인지 바로 알 수 있어야 한다.

```python
# 나쁜 예
a = 63          # a가 무엇인지 모른다
x = 1           # x가 무엇인지 모른다

# 좋은 예
patient_age = 63        # 환자의 나이임을 바로 안다
has_heart_disease = 1   # 심장병 여부임을 바로 안다
```

---

### 변수의 타입 확인 — `type()`

파이썬의 모든 값에는 **종류(타입)**가 있다. `type()` 함수로 확인한다.

```python
age = 63
name = "Alice"
height = 175.3
has_disease = True

print(type(age))      # <class 'int'>
print(type(name))     # <class 'str'>
print(type(height))   # <class 'float'>
print(type(has_disease))  # <class 'bool'>
```

## 2. 숫자형(Numeric Types) — 파이썬의 계산기

### 정수(int)와 실수(float)

파이썬에는 두 가지 기본 숫자 타입이 있다.

| 타입 | 영어 | 예시 | 설명 |
|:---|:---|:---|:---|
| **정수형**(int) | integer | `1`, `25`, `-10`, `0` | 소수점 없는 수 |
| **실수형**(float) | floating point | `1.0`, `3.14`, `-0.5` | 소수점 있는 수 |

```python
# 정수형 (int)
patient_count = 299
age = 63

# 실수형 (float)
bmi = 22.5
ejection_fraction = 38.0
```

---

### 사칙연산과 특수 연산자

```python
a = 10
b = 3

print(a + b)   # 13  --> 덧셈
print(a - b)   # 7   --> 뺄셈
print(a * b)   # 30  --> 곱셈
print(a / b)   # 3.3333...  --> 나눗셈 (항상 float 반환!)
print(a // b)  # 3   --> 몫 (정수 나눗셈)
print(a % b)   # 1   --> 나머지
print(a ** b)  # 1000 --> 거듭제곱 (10의 3제곱)
```

> **주의!** `/`(나눗셈)은 항상 소수점을 포함한 결과를 반환한다. `10 / 2`는 `5`가 아니라 `5.0`이다.

---

### 형 변환(Type Conversion) — 타입을 바꾸는 방법

때로는 숫자를 문자열로, 또는 문자열을 숫자로 바꿔야 한다.

```python
# 정수 --> 실수
x = int(3.9)    # 결과: 3  (소수점 이하 버림! 반올림이 아니다)
y = int(3.1)    # 결과: 3

# 문자열 --> 정수
age_str = "63"
age_int = int(age_str)   # 결과: 63 (이제 계산 가능)

# 정수 --> 문자열
num = 100
num_str = str(num)   # 결과: "100"
```

> **함정 주의!** `int(3.9)`는 `4`가 아니라 `3`이다. 파이썬은 소수점 이하를 **버린다**. 반올림하려면 `round()` 함수를 써야 한다.

```python
print(round(3.9))   # 4  --> 반올림
print(int(3.9))     # 3  --> 버림
```

---

### 유용한 숫자 함수들

```python
print(abs(-10))        # 10  --> 절댓값
print(max(3, 7, 1))    # 7   --> 최댓값
print(min(3, 7, 1))    # 1   --> 최솟값
print(round(3.567, 2)) # 3.57 --> 소수점 둘째 자리까지 반올림
```

## 3. 문자열(String) — 텍스트를 다루는 방법

### 문자열이란

**문자열**(String)이란 따옴표(`'` 또는 `"`)로 감싸인 텍스트이다.

```python
name = "홍길동"
disease = 'heart failure'
message = "환자 ID: 001"
```

작은따옴표(`'`)와 큰따옴표(`"`)는 둘 다 사용할 수 있다. 단, **시작과 끝을 같은 종류로** 써야 한다.

```python
# 올바른 예
s1 = "Hello"
s2 = 'World'

# 오류 예
s3 = "Hello'  # --> SyntaxError!
```

---

### 문자열 연산 — 이어 붙이기와 반복

```python
first = "심장"
second = "병원"

# 이어 붙이기 (+)
result = first + second
print(result)   # 심장병원

# 반복 (*)
echo = "안녕 " * 3
print(echo)     # 안녕 안녕 안녕
```

> **함정 주의!** 숫자처럼 보여도 따옴표 안에 있으면 문자열이다.
> `"63" + "10"`은 `73`이 아니라 `"6310"`이다!

```python
a = "63"
b = "10"
print(a + b)    # 6310  (이어 붙이기!)
print(int(a) + int(b))  # 73  (숫자로 변환 후 덧셈)
```

---

### 문자열 인덱싱(Indexing) — 글자 하나 꺼내기

문자열의 각 글자에는 **번호(인덱스)**가 붙어 있다.

```
문자열:  H  e  l  l  o
인덱스:  0  1  2  3  4
음수:   -5 -4 -3 -2 -1
```

```python
word = "Hello"

print(word[0])    # H  --> 첫 번째 글자
print(word[4])    # o  --> 다섯 번째 글자
print(word[-1])   # o  --> 마지막 글자
print(word[-2])   # l  --> 뒤에서 두 번째 글자
```

> **파이썬의 인덱스는 0부터 시작한다!** 이것이 처음에는 어색하지만 금방 익숙해진다.

---

### 문자열 슬라이싱(Slicing) — 일부 잘라내기

`[시작:끝]` 형태로 문자열의 일부를 잘라낼 수 있다. **끝 인덱스는 포함되지 않는다.**

```python
word = "Hello World"

print(word[0:5])    # Hello  --> 0번부터 4번까지
print(word[6:11])   # World  --> 6번부터 10번까지
print(word[:5])     # Hello  --> 처음부터 4번까지
print(word[6:])     # World  --> 6번부터 끝까지
print(word[::2])    # HloWrd --> 2칸씩 건너뛰기
print(word[::-1])   # dlroW olleH --> 문자열 뒤집기
```

---

### 문자열 메서드(Method) — 문자열을 변형하는 도구들

메서드(Method)는 점(`.`)을 붙여서 사용하는 **문자열 전용 함수**이다.

```python
s = "  Hello, World!  "

# 대소문자 변환
print(s.upper())        # "  HELLO, WORLD!  "
print(s.lower())        # "  hello, world!  "

# 공백 제거
print(s.strip())        # "Hello, World!"  --> 양쪽 공백 제거
print(s.lstrip())       # "Hello, World!  " --> 왼쪽만
print(s.rstrip())       # "  Hello, World!" --> 오른쪽만

# 찾기와 바꾸기
print(s.replace("Hello", "Hi"))   # "  Hi, World!  "
print(s.find("World"))            # 9  --> 위치 반환 (없으면 -1)

# 분리와 결합
csv_line = "홍길동,25,남,서울"
parts = csv_line.split(",")
print(parts)   # ['홍길동', '25', '남', '서울']

print(" / ".join(parts))   # 홍길동 / 25 / 남 / 서울

# 포함 여부 확인
print("World" in s)     # True
print("Python" in s)    # False

# 길이
print(len(s))           # 17
```

### f-문자열(f-string) — 가장 세련된 출력 방법

파이썬 3.6부터 도입된 **f-문자열**(f-string)은 문자열 안에 변수를 직접 넣는 방법이다.

```python
patient_id = "P001"
age = 63
ejection_fraction = 38.0
status = "위험"

# 기존 방식 (번거로움)
print("환자 " + patient_id + ", 나이: " + str(age))

# f-string 방식 (깔끔!)
print(f"환자 {patient_id}, 나이: {age}세, 박출률: {ejection_fraction}%")
# 출력: 환자 P001, 나이: 63세, 박출률: 38.0%

# 숫자 서식 지정
print(f"박출률: {ejection_fraction:.1f}%")   # 소수점 1자리
print(f"나이: {age:>5}세")                    # 5칸 오른쪽 정렬
```

## 4. 실전 적용 — 심장병 환자 데이터 읽기

이번 주에 배운 내용을 심장병 데이터에 적용해 본다.

```python
# 심장병 환자 한 명의 데이터를 변수로 저장
patient_id = "P001"
age = 63
ejection_fraction = 38      # 박출률 (%)
serum_creatinine = 1.1      # 혈청 크레아티닌 (mg/dL)
diabetes = 1                # 당뇨 여부 (1=있음, 0=없음)
DEATH_EVENT = 1             # 사망 여부 (1=사망, 0=생존)

# 정보 출력
print(f"환자 정보 보고서")
print(f"{'='*30}")
print(f"환자 ID     : {patient_id}")
print(f"나이        : {age}세")
print(f"박출률      : {ejection_fraction}%")
print(f"혈청크레아티닌: {serum_creatinine:.1f} mg/dL")
print(f"당뇨 여부   : {'있음' if diabetes == 1 else '없음'}")
print(f"생존 여부   : {'사망' if DEATH_EVENT == 1 else '생존'}")
```

출력 결과:
```
환자 정보 보고서
==============================
환자 ID     : P001
나이        : 63세
박출률      : 38%
혈청크레아티닌: 1.1 mg/dL
당뇨 여부   : 있음
생존 여부   : 사망
```

## 5. 핵심 정리

### 변수(Variable)
- 값을 저장하는 이름표이다
- `=`는 "오른쪽 값을 왼쪽에 저장하라"는 명령이다
- `type()`으로 타입을 확인한다

### 숫자형(Numeric)
| 연산 | 기호 | 예시 |
|:---|:---|:---|
| 덧셈 | `+` | `3 + 2 = 5` |
| 뺄셈 | `-` | `3 - 2 = 1` |
| 곱셈 | `*` | `3 * 2 = 6` |
| 나눗셈 | `/` | `3 / 2 = 1.5` |
| 몫 | `//` | `3 // 2 = 1` |
| 나머지 | `%` | `3 % 2 = 1` |
| 거듭제곱 | `**` | `3 ** 2 = 9` |

### 문자열(String)
| 기능 | 코드 | 결과 |
|:---|:---|:---|
| 이어붙이기 | `"a" + "b"` | `"ab"` |
| 반복 | `"a" * 3` | `"aaa"` |
| 첫 글자 | `s[0]` | 첫 번째 글자 |
| 마지막 글자 | `s[-1]` | 마지막 글자 |
| 슬라이싱 | `s[1:4]` | 1~3번 글자 |
| 대문자 | `s.upper()` | 모두 대문자 |
| 공백 제거 | `s.strip()` | 양쪽 공백 제거 |

## 6. 자주 묻는 질문(FAQ)

**Q. `=`와 `==`는 어떻게 다른가?**
A. `=`는 **저장** 연산자이고, `==`는 **비교** 연산자이다. `age = 25`는 "age에 25를 저장해라"이고, `age == 25`는 "age가 25인가?"라고 묻는 것이다.

**Q. 문자열에서 인덱스가 왜 0부터 시작하는가?**
A. 컴퓨터 메모리 주소 계산의 역사적 이유이다. 처음에는 어색하지만, 모든 프로그래밍 언어가 이 방식을 사용하므로 익숙해지는 것이 중요하다.

**Q. `int(3.9)`가 4가 아닌 3인 이유는?**
A. `int()`는 소수점 이하를 **버린다**. 반올림이 아니다. 반올림하려면 `round(3.9)`를 사용해야 한다.

**Q. f-string 안에서 계산도 되는가?**
A. 된다. `f"{2 + 3}"` --> `"5"`, `f"{name.upper()}"` --> 대문자 이름
